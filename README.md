1. Clone this repository to your device:
   ```bash
   git clone https://github.com/Enver-Onur-Cogalan/Expense-Tracker-.git
   ```

2. Navigate into the project directory:
   ```bash
   cd ExpenseTracker-RN
   ```

3. Install dependencies:
   ```bash
   yarn install
   ```

4. For iOS:
   ```bash
   cd ios
   pod install
   cd ..
   ```

5. To run on Android:
   ```bash
   yarn android
   ```

6. To run on iOS:
   ```bash
   yarn ios
   ```
